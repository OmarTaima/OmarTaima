## 🚀 Featured Project

### Job Application Maker

A modern **Applicant Tracking System (ATS)** built with **React** and **TypeScript** that streamlines the complete recruitment lifecycle—from job posting and applicant management to interviews, offers, and contract generation.

### Highlights

- 🌍 Multi-company architecture
- 👥 Complete applicant tracking workflow
- 📅 Interview scheduling & evaluation
- 📄 Job offers and contract management
- 📧 Email template system with Resend API
- 🔐 Role-based access control
- 📊 Analytics dashboard
- 🌐 English & Arabic (RTL/LTR)
- 🌙 Dark mode support

### Tech Used

`React` • `TypeScript` • `Vite` • `TanStack React Query` • `Redux Toolkit` • `Material UI` • `Tailwind CSS` • `React Router` • `Axios`

🔗 **Repository:** *https://github.com/Saber-Group-EG/Job-Application-Maker*
